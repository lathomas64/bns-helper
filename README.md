# bns-helper
expo rock paper scissors for BNS games.

initial setup:
npx create-expo-app rps
cd rps
npx expo start

npm install -g eas-cli


# deploy
npm run deploy

# start local
npx expo start

# generate an apk
eas build --profile development --platform android
